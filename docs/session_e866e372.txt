var ErrEmailExists        = errors.New("email already exists")

why adding ca-certification in dockerfile
---
[ASSISTANT]
The app makes outbound HTTPS calls to the Central Weather Bureau API. Alpine's base image ships with no CA bundle, so TLS verification would fail with a certificate error. `ca-certificates` provides the trusted root CAs needed to validate the server's certificate.

`tzdata` is there for correct timezone handling in time parsing — MySQL returns datetimes and the DSN uses `loc=UTC`.
